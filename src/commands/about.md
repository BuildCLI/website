
# `about`

## Description

Displays information about BuildCLI.

## Usage

```bash
buildcli about
```

## Options

| Option          | Description                                 |
|-----------------|---------------------------------------------|
| `-h, --help`    | Display help information about the command. |
| `-V, --version` | Display the version of the command.         |

## Examples

### Example 1

Display information about BuildCLI.

```bash
buildcli about
```
