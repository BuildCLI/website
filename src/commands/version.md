# `version`

## Description

Displays the version of BuildCLI.

## Usage

```bash
buildcli version
```

## Options

| Option          | Description                                 |
|-----------------|---------------------------------------------|
| `-h, --help`    | Display help information about the command. |
| `-V, --version` | Display the version of the command.         |

## Examples

### Example 1

Display the version of BuildCLI.

```bash
buildcli version
```
